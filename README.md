# Cozeva

Assignment for the MERN Stack Developer.

## Requirements

1. nodeJS >=14.15.4
2. npm >=6.14.10

## Steps to init project

1.  First of all, install the dependencies with npm:

    `npm install` in the root directory

2.  ### Start the server

    `npm start` in the root directory

3.  Now load the app

    ```javacript
    localhost:3000
    ```
